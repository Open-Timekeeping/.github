# Open Timekeeping, community health

This is GitHub's special `.github` repository for the **Open Timekeeping** organization. It holds the organization profile and the community-health files that GitHub auto-applies across our other repos.

- [`profile/README.md`](./profile/README.md), the page rendered at <https://github.com/Open-Timekeeping>. Entry point for the organization.
- [`profile/repos.md`](./profile/repos.md), repo family, naming hierarchy, adapter-plugin strategy.
- [`profile/languages.md`](./profile/languages.md), per-layer language choices and rationale.
- Future additions: org-wide `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md`, default issue and pull-request templates, and workflow templates.

For the actual code, see the projects linked from the [organization profile](./profile/README.md). The canonical design spec lives in [spec](https://github.com/Open-Timekeeping/spec).
