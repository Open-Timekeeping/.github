# Open Timekeeping — community health

This is GitHub's special `.github` repository for the **Open Timekeeping** organization. It holds the organization profile and the community-health files that GitHub auto-applies across our other repos.

- [`profile/README.md`](./profile/README.md) — the page rendered at <https://github.com/Open-Timekeeping>.
- Future additions: org-wide `CODE_OF_CONDUCT.md`, `CONTRIBUTING.md`, `SECURITY.md`, default issue and pull-request templates, and workflow templates.

For the actual code, see the projects linked from the [organization profile](./profile/README.md). The canonical design spec lives in [open-timing-fabric](https://github.com/Open-Timekeeping/open-timing-fabric) under `AGENTS.md`.
